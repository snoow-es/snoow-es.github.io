Política de Privacidad

Gracias por visitar nuestro sitio web. Esta Política de Privacidad describe cómo gestionamos la información en nuestro sitio web, que es un proyecto personal, sin ánimo de lucro y alojado en GitHub Pages.

1. Sobre Nosotros

Este sitio web es un proyecto personal sin ánimo de lucro y no pertenece a ninguna organización comercial o empresarial. Nuestro único objetivo es ofrecer contenido informativo y de interés personal al público general.

2. Recopilación y uso de información personal

No recopilamos, almacenamos, ni tratamos ningún dato personal o identificativo sobre nuestros visitantes. No utilizamos cookies, herramientas de seguimiento ni tecnologías similares para identificar o rastrear a nuestros usuarios.

3. GitHub Pages

Este sitio web está alojado en GitHub Pages, un servicio proporcionado por GitHub. GitHub puede recopilar información técnica mínima como parte de su funcionamiento habitual (por ejemplo, dirección IP o información técnica para mantener la seguridad del servicio). Para más información, puede consultar la Política de Privacidad de GitHub.

4. Enlaces externos

Este sitio puede contener enlaces a otros sitios web externos. No somos responsables del contenido ni de las prácticas de privacidad de estos sitios externos. Le recomendamos revisar sus respectivas políticas de privacidad.

5. Seguridad

Aunque no almacenamos información personal alguna, hacemos nuestro mejor esfuerzo por mantener nuestro sitio web seguro y libre de amenazas digitales.

6. Cambios en esta Política de Privacidad

Nos reservamos el derecho a actualizar esta política de privacidad en cualquier momento. La última versión siempre estará disponible en esta página con la fecha de última actualización claramente indicada.

7. Contacto

Si tiene preguntas, comentarios o inquietudes sobre esta política de privacidad o cualquier aspecto de este sitio web, no dude en ponerse en contacto con nosotros escribiendo a hello@snoow.es.
